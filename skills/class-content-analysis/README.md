# Class Content Analysis

Reviews your own readings, slides, and assignments the way an outside reviewer would: coverage gaps, wasteful overlap, reading-level mismatches, assessment alignment, and how the course has drifted since last time.

Use it when you want the content review you would never find time to schedule yourself, before a term starts or after it ends.

## What's in this folder

- `SKILL.md`: the five checks, the prompt, the category-weighting add-on, a worked example, what to check, and the guardrail. Open this to run the skill.

For pulling materials directly from your LMS instead of pasting them, see `../../guides/canvas-lms.md` or `../../guides/other-lms.md`.
