# Start Here

The front door for a teacher who just met this toolkit: orients them in two sentences, asks a few questions one at a time, offers to ingest their materials, and routes them to the one skill that fixes their biggest time-sink.

Use it whenever someone has just installed these skills or pointed an agent at this repo and wants to get started, or says "help me get started" or "onboard me."

## What's in this folder

- `SKILL.md`: the prompt, the routing table, how to keep ingestion friction-free, and the guardrail. Open this to run the skill.

This is an operational skill; it never touches student data or a gradebook itself. It hands off to whichever skill fixes the teacher's stated problem.
