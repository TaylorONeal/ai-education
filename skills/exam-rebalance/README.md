# Exam Rebalance

Pressure-tests an exam before anyone takes it: rates difficulty and coverage, simulates six named student personas taking it, and compares the whole thing against every prior version you have.

Use it when you are finalizing a midterm, final, or quiz and want to catch an unfair or lopsided test while you can still fix it.

## What's in this folder

- `SKILL.md`: the three-part check (difficulty and coverage, persona simulation, prior-version comparison), the six personas, the calibration table, the two levers that fix most spreads, what to check, and the guardrail. Open this to run the skill.

For pulling the current exam and prior versions directly from your LMS, see `../../guides/canvas-lms.md` or `../../guides/other-lms.md`. For assembling years of past exams into one comparison, see `../../guides/automation.md`.
