# Grading Assistant

Applies the rubric you wrote consistently across a whole stack, and catches the student who understood the concept but phrased it in their own words, or a second language, and would otherwise lose points for wording instead of knowledge.

Use it when you are grading short answers, essays, or open-response work and want a second set of eyes that credits understanding instead of penalizing phrasing.

## What's in this folder

- `SKILL.md`: how to write a rubric a machine can apply, the prompt, a worked example, the two ways to finish (spreadsheet only, or grades into Canvas), what to check, and the guardrail. Open this to run the skill.
- `grading-template.xlsx`: the blank spreadsheet the scored output drops into (a Grading sheet, a Rubric-bands sheet, and a How-to sheet).

For entering grades into Canvas or another LMS, see `../../guides/canvas-lms.md` or `../../guides/other-lms.md`.
