# Participation Scoring

Turns a gut feeling about who seemed engaged into an auditable record: every student's real activity across the channels that count for your class, scored on rules you set before you looked at a single post.

Use it when you have to grade class participation and want a record you could show a student who challenges their grade, instead of a memory of who talked the most.

## What's in this folder

- `SKILL.md`: the scoring model, the bucket table, the prompt, the name-matching gotchas, and the guardrail. Open this to run the skill.

For pulling activity from a specific chat tool or LMS, see `../../guides/chat-and-discussion.md`. For assembling a full-archive pull across the term, see `../../guides/automation.md`.
