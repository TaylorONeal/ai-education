# Announcement Writer

Drafts class announcements in your own voice, learned from your real past posts, then runs a pass that strips the machine tells before anything reaches a student.

Use it for weekly updates, exam details, schedule changes, welcome and goodbye posts, TA introductions, and extra-credit notices.

## What's in this folder

- `SKILL.md`: how to capture your voice as a reusable style guide, the category templates, the prompt, the AI-tell cleanup pass, what to check, and the guardrail. Open this to run the skill.

For posting automatically through a specific platform, see `../../guides/canvas-lms.md` or `../../guides/chat-and-discussion.md`.
