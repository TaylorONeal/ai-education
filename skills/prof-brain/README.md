# Prof Brain

Ingests your scattered course material (local folders, Google Drive, LMS reading pages, syllabus, slides, past exams) into one organized folder of Markdown notes with an index, so you and every other skill read from one grounded source instead of re-explaining context each session.

Use it to set up or refresh your course knowledge base, and optionally sync it to Notion or an Obsidian vault.

## What's in this folder

- `SKILL.md`: the note schema, the folder tree, how to ingest each kind of source, the prompt, the Notion and Obsidian sync steps, what to check, and the guardrail. Open this to run the skill.

Every other skill in the toolkit sharpens when pointed at the brain instead of pasted context; see `../../AGENTS.md` for which skill reads which corner of it.
