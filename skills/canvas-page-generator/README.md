# Course Page Generator

Turns plain text into a clean, styled course page built from a small set of reusable blocks, so you write the content and skip the formatting fight in a clunky LMS editor.

Use it for assignment pages, reading pages, and info pages, in Canvas or any other LMS.

## What's in this folder

- `SKILL.md`: the component library, what your editor will strip or mangle, the prompt, capturing your brand once, the page types, what to check, and the guardrail. Open this to run the skill.
- `brand-guidelines.md`: the template your captured palette and type get saved into, so every future page matches without retyping colors.

For pushing a finished page into a platform through its API, see `../../guides/canvas-lms.md` or `../../guides/other-lms.md`.
