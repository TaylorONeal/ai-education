# Canvas LMS Operations

The operational layer that runs Canvas reliably: authentication, pushing styled pages and assignment HTML, entering grades without losing them, and editing quizzes and question banks.

Use it when a task from another skill (grades, pages, submission HTML) has to land in Canvas, or any time you are working with the Canvas API, SpeedGrader, the Rich Content Editor, or quiz and question-bank changes.

## What's in this folder

- `SKILL.md`: the operations, the auth paths, what to check after each one, and the guardrail. Open this to run the skill.

The full reference, every endpoint, the large-body chunking technique, and the quiz-bank traps, is `../../guides/canvas-lms.md`. The equivalent for other platforms is `../../guides/other-lms.md`.
