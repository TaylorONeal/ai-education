# Exam Predictor

Forecasts each student's likely exam score from graded coursework, weighting assignments by how well they actually predict exam performance instead of averaging everything together.

Use it after enough coursework is graded and before the exam, to flag who is at risk while you can still reach out.

## What's in this folder

- `SKILL.md`: the predictive-weighting model, the behavioral signals that set the confidence band, the prompt, calibration against the exam-rebalance personas, what to check, and the guardrail. Open this to run the skill.

For pulling gradebook history across terms, see `../../guides/canvas-lms.md`, `../../guides/other-lms.md`, and `../../guides/automation.md`.
